# SayItBetter

AI 驱动的社媒文案助手，选中即用，一键生成。

## 功能

- 🖱️ 选中网页文本，一键生成 AI 改写内容
- 📱 支持多平台：Twitter、小红书、LinkedIn、朋友圈
- 🎨 15+ 写作风格模板
- ✨ 智能人性化处理，去除 AI 写作痕迹
- 💾 本地存储，数据不上传

## 安装

1. 下载 `.crx` 文件
2. 打开 `chrome://extensions/`
3. 开启"开发者模式"
4. 拖拽 `.crx` 文件安装

## 使用

1. 配置 API Key（OpenAI/Anthropic/智谱/DeepSeek）
2. 选中网页文本
3. 按 `Ctrl+Shift+Q` (Mac: `Cmd+Shift+Q`)
4. 选择平台和风格，生成内容

## 开发

```bash
npm install
npm run dev
npm run build
```

## 隐私

API Key 仅存储在本地浏览器中。
