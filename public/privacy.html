SayItBetter 隐私政策

最后更新：2026 年 4 月 19 日

概述

SayItBetter 是一款浏览器扩展，帮助用户将选中的文本通过 AI 改写为适合社交媒体平台的文案。我们高度重视您的隐私。

数据收集

SayItBetter 不收集任何个人数据。我们不使用 Cookie、不使用分析工具、不使用遥测技术。

数据存储

所有设置（包括您自行配置的 API Key）和历史记录仅存储在浏览器本地（chrome.storage.local 和 IndexedDB），不会发送到任何第三方服务器。

API 调用

当您使用文本改写功能时，选中的文本将直接从您的浏览器发送到您选择的 AI 服务提供商（如 OpenAI、Anthropic、DeepSeek 等）。API 调用由您自行配置的 API Key 驱动，我们不做任何中间处理或转发。

第三方服务

SayItBetter 可能连接以下第三方 AI 服务（取决于您的配置）：

  • OpenAI (api.openai.com)
  • Anthropic (api.anthropic.com)
  • 阿里云通义千问 (dashscope.aliyuncs.com)
  • 智谱 GLM (open.bigmodel.cn)
  • 百度文心 (qianfan.baidubce.com)
  • Moonshot Kimi (api.moonshot.cn)
  • DeepSeek (api.deepseek.com)
  • 火山引擎 (ark.cn-beijing.volces.com)

各服务的隐私政策请参阅其官方网站。

内容脚本

SayItBetter 仅在您主动选中文本并触发操作时才会处理页面内容。扩展不会在后台监控或记录您的浏览行为。

权限说明

  • storage — 存储您的设置和历史记录
  • activeTab — 获取当前标签页选中的文本
  • clipboardWrite — 将生成的文案复制到剪贴板
  • alarms — 定期清理过期历史记录
  • 可选主机权限 — 仅在您配置对应 AI 服务时按需请求

变更通知

如果本隐私政策发生变更，我们将在此页面更新最新日期。

联系方式

如有隐私相关问题，请通过 Chrome Web Store 扩展页面联系开发者。
